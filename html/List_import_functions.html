Import of topology files

  1. import_itp(filename) % Import a Gromacs .itp topology file

Import of data files

  1. import_xvg(filename) % Import a Gromacs .xvg file.
  2. import_gmx_energy(filename) % Import a Gromacs energy file.
  3. import_bar(filename) % Import a BAR file.
  4. import_dat(filename) % Import a .dat file.
  5. import_red_charges(filename) % Import reduced charges from a file.
  6. import_ddec_charges(filename) % Import DDEC charges from a file.

Import of structure files

  1. import_atom_car(filename,varargin) % Import .car files from Hendrik Heinz INTERFACE force field distribution, then write out a Gromacs molecular topology file (.itp) and a new .pdb file.
  2. import_atom_gro(filename) % Import .gro files into the atom struct.
  3. import_atom_gro_fscanf(filename) % Import .gro files using fscanf (alternative method).
  4. import_atom_gro_octave(filename) % Import .gro files using Octave-compatible code.
  5. import_atom_mol2(filename) % Import .mol2 files into the atom struct.
  6. import_atom_pdb(filename) % Import .pdb files into the atom struct.
  7. import_atom_pqr(filename) % Import .pqr files into the atom struct.
  8. import_atom_xyz(filename) % Import an .xyz file into the atom struct.
  9. import_atom_poscar(filename) % Import a VASP POSCAR file into the atom struct.
  10. import_atom(filename) % Import a .xyz, .gro, or .pdb file into a structure variable called atom.
  11. import_xyz(filename) % Import an .xyz file. Atom types should be made of letters, not numbers. Use import_atom_xyz instead.

Import of trajectory files

  1. import_gro_traj(filename,varargin) % Import a structure and a .gro trajectory file.
  2. import_xyz_traj(filenametraj) % Import a structure and an .xyz trajectory file.
  3. import_mc_pdb_traj(filename,varargin) % Import a structure and a .pdb trajectory file, handling changing numbers of particles.
  4. import_pdb_traj(filename,varargin) % Import a structure and a .pdb trajectory file.
  5. import_traj(filenameconf,filenametraj) % Import a structure and a .dcd, .trr, .xtc, .xyz, or .gro trajectory file.
  6. import_trr(filenameconf,filenametraj) % Import a structure and a .trr trajectory file.
  7. import_trrv2(filenameconf,filenametraj) % Import a structure and a .trr trajectory file (version 2).
  8. import_xtc(filenameconf,filenamextc) % Import a structure and an .xtc file.
  9. import_xtcv2(filenameconf,filenamextc) % Import a structure and an .xtc file (version 2).

Miscellaneous

  1. import_ave_gro(filename) % Import an averaged structure from a .gro trajectory.
  2. import_mclf(filename) % Import a multi-configurational London force (MCLF) file.
  3. import_mclf_C6(filename) % Import MCLF C6 dispersion parameters.
  4. import_mclf_C8(filename) % Import MCLF C8 dispersion parameters.
  5. import_mclf_C10(filename) % Import MCLF C10 dispersion parameters.
  6. import_mclf_dispersion(filename) % Import dispersion parameters for MCLF.
  7. import_cp2k(filename) % Import a CP2K output file.
  8. import_cp2k_resp(filename) % Import RESP charges from CP2K.
