Custom topology tools

minff, with atomtypes by MHolmboe

  1. minff_atom(atom,Box_dim,varargin) % Assign minff atom types with edge healing.
  2. charge_minf_atom(atom,Box_dim,varargin) % Set the charge for MINFF atom types.

Clayff, with atomtypes by MHolmboe

  1. charge_atom(atom,Box_dim,ffname,watermodel,varargin) % Charge the atom according to Clayff or Interface force fields.
  2. charge_clayff_2004_atom(atom,Box_dim,varargin) % Set the charge for the original Clayff atom types from the Cygan et al., 2004 paper.
  3. charge_clayff_atom(atom,Box_dim,varargin) % Set the charge for Clayff atom types.
  4. charge_opls_go_atom(atom,Box_dim,varargin) % Set the charge for specific OPLS atom types.
  5. check_clayff_2004_charge(atom) % Check the charge of the original Clayff atom types.
  6. check_clayff_charge(atom) % Check the charge of the Clayff atom types.
  7. check_clayff_H2Odens(atom,Box_dim) % Check the approximate water density for a Clayff system.
  8. check_H2Odens(atom,Box_dim) % Compute the water density.
  9. clayff_2004_atom(atom,Box_dim,varargin) % Assign the original Clayff atom types by Cygan et al., 2004, with edge healing.
  10. clayff_2004_param(Atom_label,varargin) % Hold ion and Clayff atom type parameters for the original Clayff force field.
  11. clayff_atom(atom,Box_dim,varargin) % Assign Clayff atom types with edge healing.
  12. clayff_param(Atom_label,varargin) % Hold ion and Clayff atom type parameters.
  13. clayff210_atom(atom,Box_dim,varargin) % Assign modified Clayff atom types, with edge healing.
  14. clayff211_atom(atom,Box_dim,varargin) % Assign modified Clayff atom types (faster version), with edge healing.
  15. tweak_charge_atom(atom) % Tweak the charge of the atom struct to correct rounding errors.

INTERFACE from Heinz 2005, 2013, with atomtypes by MHolmboe

  1. charge_atom(atom,Box_dim,ffname,watermodel,varargin) % Charge the atom according to Clayff or Interface force fields.
  2. interface_atom(atom,Box_dim,varargin) % Assign atoms according to the Interface atom types, with modifications for edges.
  3. interface_param(Atom_label,water_model) % Hold extended INTERFACE force field parameters.
  4. interface15_atom(atom,Box_dim,varargin) % Assign atoms according to the Interface 1.5 atom types, with modifications for edges.
  5. interface15_param(Atom_label,water_model) % Hold extended INTERFACE 1.5 force field parameters.
  6. charge_interface_atom(atom,Box_dim,varargin) % Set the charge for Interface atom types.
  7. charge_interface15_atom(atom,Box_dim,varargin) % Set the charge for Interface 1.5 atom types.
  8. check_interface_charge(atom) % Check the charge of the INTERFACE atom types.
  9. check_interface15_charge(atom) % Check the charge of the INTERFACE 1.5 atom types.
  10. interface15_silica_atom(atom,Box_dim,varargin) % Assign atom types for the Interface 1.5 force field, specific to silica.
  11. tweak_charge_atom(atom) % Tweak the charge of the atom struct to correct rounding errors.

Graphene oxide modeled with OPLS/aa

  1. opls_go_atom(atom,Box_dim,rmin,rlarge) % Smear out the charge around -OH and epoxide groups in graphene oxide.
  2. oplsaa_go_param(Atom_label,water_model) % Hold the extended OPLS-AA force field parameters for graphite oxide.
  3. charge_opls_go_atom(atom,Box_dim,varargin) % Set the charge for specific OPLS atom types.

Writing topology files for Lammps (Clayff) and Gromacs (Clayff/INTERFACE)

  1. write_atom_itp(atom,Box_dim,filename_out,varargin) % Create and print a Gromacs .itp file for Clayff or Interface force fields.
  2. write_atom_lmp(atom,Box_dim,filename_out,varargin) % Create and print a LAMMPS data file (.lj) for Clayff systems.
  3. write_atom_oplsaa_go_itp(atom,Box_dim,filename_out,varargin) % Create and print a Gromacs .itp file for OPLS-AA or GO systems.

Bonded and Nonbonded Parameters

  1. bonded_parameters(atom,varargin) % Define bonded parameters for atoms.
  2. nonbonded_parameters(atom,varargin) % Define nonbonded parameters for atoms.
  3. nonbonded_ff(atom,varargin) % Defines nonbonded forcefield parameters.

Lennard-Jones and Coulomb Potentials

  1. buckinghamcoul(atom,Box_dim,varargin) % Calculates interactions using the Buckingham potential and Coulombic forces.
  2. ljcoul_12_6(atom,Box_dim,varargin) % Handles Lennard-Jones 12-6 potential along with Coulombic interactions.
  3. ljcoul_C12C6C4(atom,Box_dim,varargin) % Lennard-Jones and Coulomb potential with C12, C6, C4 terms.
  4. ljcoul_C12C6(atom,Box_dim,varargin) % Lennard-Jones and Coulomb potential with C12, C6 terms.
  5. ljcoul(atom,Box_dim,varargin) % General Lennard-Jones and Coulomb interaction function.
  6. ljcoul_2x2x(atom,Box_dim,varargin) % Lennard-Jones and Coulomb potential with 2x factors.
  7. ljcoul_2x(atom,Box_dim,varargin) % Variation of Lennard-Jones Coulomb with 2x factors.

Objective Functions and Force Calculations

  1. buckinghamcoul_objective_func(atom,Box_dim,varargin) % Objective function for fitting Buckingham and Coulomb potentials.
  2. ljcoul_objective_func(atom,Box_dim,varargin) % Objective function for Lennard-Jones and Coulomb potentials.
  3. ljcoul_2x_objective_func(atom,Box_dim,varargin) % Objective function for 2x Lennard-Jones and Coulomb potentials.
  4. ljcoul_force(atom,Box_dim,varargin) % Computes forces based on Lennard-Jones and Coulomb potentials.
  5. ljcoul_2x_force(atom,Box_dim,varargin) % Force calculation involving 2x Lennard-Jones and Coulomb potentials.
  6. ljcoul_force_C12C6C4(atom,Box_dim,varargin) % Force calculation for Lennard-Jones potential with C12, C6, C4 terms.
  7. ljcoul_force_objective_func(atom,Box_dim,varargin) % Objective function for force calculations with Lennard-Jones and Coulomb potentials.

Automated Fitting Tools

  1. autofit_C6C8C10xljcoul(atom,Box_dim,varargin) % Automated fitting for Lennard-Jones Coulombic parameters with C6, C8, C10 terms.
  2. autofit_2xljcoul_func(atom,Box_dim,varargin) % Function for automatic fitting of 2x Lennard-Jones and Coulomb parameters.
  3. autofit_2xljcoul_batch(atom,Box_dim,varargin) % Batch fitting for 2x Lennard-Jones and Coulomb potentials.
  4. autofit_2xljcoul(atom,Box_dim,varargin) % Automated fitting of 2x Lennard-Jones and Coulomb parameters.
  5. autofit_geometric2LB(atom,Box_dim,varargin) % Automated geometric fitting for Lennard-Jones potential with Born-Mayer interactions.
  6. autofit_buckcoul(atom,Box_dim,varargin) % Automated fitting for Buckingham and Coulomb potentials.
  7. autofit_force_2xljcoul(atom,Box_dim,varargin) % Automated fitting of force parameters for 2x Lennard-Jones and Coulomb interactions.
  8. autofit_ljcoul(atom,Box_dim,varargin) % Automated fitting for Lennard-Jones and Coulombic interactions.

Miscellaneous

  1. print_top(atom,Box_dim,varargin) % Prints or generates topology-related data.
  2. import_ff_table(filename,varargin) % Imports forcefield parameter tables.
  3. change_top(atom,Box_dim,varargin) % Modifies the topology file or its parameters.
  4. mass_atom_clayff(atom) % Defines or calculates masses for Clayff atoms.
  5. harmonic_bond(atom,Box_dim,varargin) % Defines or computes harmonic bond forces or parameters.
  6. smear_charge(atom,Box_dim,varargin) % Distributes charge across atoms, possibly using charge smearing techniques.
  7. check_interface_charge(atom) % Checks charges for INTERFACE atom types.
  8. check_interface15_charge(atom) % Checks charges for INTERFACE 1.5 atom types.
